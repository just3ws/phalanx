# Quick Reference

<p class="print-note">Use browser print for a paper table aid. This page is formatted to stay compact and readable when printed.</p>

<section class="two-col">
  <article class="card">
    <h2>Setup</h2>
    <ul class="quick-list">
      <li>Two players.</li>
      <li>Standard suited decks (ideal: one deck per player).</li>
      <li>Use Ace, numbered cards, and face cards.</li>
      <li>Joker optional, expansion-style content only.</li>
    </ul>
  </article>

  <article class="card">
    <h2>Turn Loop</h2>
    <ol class="quick-list">
      <li>Choose front-row attacker.</li>
      <li>Attack same column.</li>
      <li>Resolve overflow: front -> back -> player LP.</li>
      <li>Reinforce if column breaks.</li>
      <li>Pass turn.</li>
    </ol>
  </article>
</section>
