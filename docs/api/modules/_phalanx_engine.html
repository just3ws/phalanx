    Module @phalanxduel/engine

    The Phalanx Duel Game Engine provides the core deterministic rules for tactical card combat.

    Remarks

    This package exposes pure functions for game rule evaluation. Every transition is side-effect free: no I/O, no randomness (RNG is injected), and no transport. This architecture ensures every game is 100% replayable and verifiable.
